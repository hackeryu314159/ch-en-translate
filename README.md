# ch-en-translate
中英文翻譯工具
# 介紹
這是一個中英文翻譯程式，
第一個輸入格輸入en/ch來決定我們需要輸入中文還英文，
接著第二行寫出您需要翻譯的東西，
接著送出過後我們就會將我們的東西輸出，
輸出第一格放翻譯後的字，
第二格放翻譯後的音檔。
